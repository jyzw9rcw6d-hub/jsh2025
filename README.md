# JSH2025 高血圧専門問題集

日本高血圧学会ガイドラインに基づいた高血圧専門問題集です。専門医試験レベルから超高難度の問題を網羅しています。

## 特徴

- 🎓 **専門医試験対応**: 難易度が★1から★10まで段階的に設定
- 📚 **充実した問題数**: 100問以上の問題を収録
- 🔄 **復習機能**: 間違えた問題のみを重点的に復習可能
- 📱 **レスポンシブデザイン**: スマートフォンやタブレットにも対応
- 🎯 **詳細な解説**: すべての問題に専門的な解説を付記

## 主なカテゴリー

- 血圧評価・診断
- 家庭血圧
- 降圧薬選択
- CKD・腎疾患
- 二次性高血圧
  - 原発性アルドステロン症
  - 腎血管性高血圧
  - 褐色細胞腫
- 糖尿病合併高血圧
- 高齢者高血圧
- 睡眠時無呼吸症候群
- 心不全合併高血圧
- 治療抵抗性高血圧

## 使用方法

### オンライン利用

1. このリポジトリをGitHub Pagesで公開
2. `index.html`にアクセス
3. 「開始」ボタンをクリックして問題を解き始める

### ローカル利用

```bash
# リポジトリをクローン
git clone https://github.com/jyzw9rcw6d-hub/jsh2025.git

# フォルダに移動
cd jsh2025

# ブラウザでindex.htmlを開く
```

## 機能説明

### 1. 通常モード
- 問題がランダムに表示されます
- 各問題に対して4つの選択肢から選びます
- 回答後、正解・不正解と詳細な解説が表示されます
- 成績が記録され、最終的な正答率が表示されます

### 2. 復習モード
- 通常モードで間違えた問題のみを復習
- 弱点を集中的に克服できます
- 「間違えた問題を復習」ボタンから起動

## カスタマイズ

### 問題を追加する場合

`js/questions.js`に以下の形式で追加してください：

```javascript
{
  id: 101,
  category: "カテゴリー名",
  difficulty: "★★★★★",
  question: "問題文",
  choices: [
    "選択肢1",
    "選択肢2",
    "選択肢3",
    "選択肢4"
  ],
  answer: 0,  // 正解の選択肢番号（0から始まる）
  explanation: "解説文"
}
```

### スタイルをカスタマイズする場合

`css/style.css`を編集してください。主な色定義はCSS変数で管理されています：

```css
:root {
    --primary-color: #12355b;      /* メインカラー */
    --success-color: #28a745;      /* 正解色 */
    --danger-color: #dc3545;       /* 不正解色 */
    /* ... その他の色 */
}
```

## トラブルシューティング

### 問題が表示されない
- ブラウザの��ンソール（F12）で エラーメッセージを確認
- `js/questions.js`が正しく読み込まれているか確認

### スタイルが反映されない
- キャッシュをクリア（Ctrl+Shift+R）
- `css/style.css`のパスが正しいか確認

### モバイルで表示がおかしい
- ビューポート設定を確認（`<meta name="viewport">`）
- ブラウザのズーム レベルをリセット
